# 🍳 scanf() : 공백만나기전 까지만 입력받는 함수

## 🟢 입력받기

prinf와 문법은 비슷하나 2번째 매개변수 인자에 & 를 붙인다.

- pointer 개념 ( 나중에 공부 )

```c
int a; //문자 1개 char
scanf("%d", &a); // 문자 1개 %c
```

## 🟢 2개 이상 입력

```c
int a; int b;
scanf("%d %d", &a, &b);
```

## 🟢 여러자리 숫자 1개씩 입력받기

ex : 숫자가 755라고 할때
%1d %1d %1d 면 7, 5, 5로 나누어서 받는다.

```c
int a; int b
scanf("%1d %2d", &a, &b);
```

## 🟢 문자열 입력

문자열 및 배열은 [0]번째 원소가 배열의 시작주소를 가지고 있기때문에 &를 붙이지 않는다.

```c
char a[20];
scanf("%s", a);
```

# 🍳 fgets() : 공백 포함 입력받는 함수

## 🟢 사용방법

```c
fgets(s, sizeof(s), stdin);
```
